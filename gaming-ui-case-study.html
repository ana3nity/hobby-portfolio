Takeaways

  • Continue to study graphic design methods such as layouts
  • Continue learning Unity
  • Interaction Design is big in the gaming industry; Player feedback with UI engages players and improves immersion
  • Including UX Research and UX Design into UI Design will improve the user flows by observing how players interact with call to actions, screens, information displayed, etc.
